# HoneyBees
Stand-alone HoneyBees mod from my main mod **[Foodies](https://modrinth.com/mod/foodies)** including most of features from 1.1 update.

Built for Minecraft **Beta 1.7.3** and **StationAPI**.

## Added Features:
- Beehive
- Wild bee nest - found in trunk of oak or birch trees
- Bees - produce honey overtime, has day-night cycle
- Honey
- Glazed food - porchop, fish and apple in honey glaze, they heal slightly more
- Honey cookies and candies
- Honeycomb block - just decorating block
- Jar - used to catch bees

## Required dependencies

- **[StationAPI](https://modrinth.com/mod/stationapi)**
- **[FarnUtil](https://modrinth.com/mod/farnutil)**
- **[Glass Config API](https://modrinth.com/mod/glass-config-api)** - Required only to change default mod config.
